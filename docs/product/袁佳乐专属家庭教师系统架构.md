# 袁佳乐专属家庭教师系统架构

生成日期：2026-06-04

## 产品定位

本项目是袁佳乐的一对一家庭教师系统。它不是题库、不是 PDF 生成器、不是短期冲刺计划，而是一个长期陪伴型学习产品。

系统目标：

- 用每日短回合练习，把当前 D 水平逐步推高。
- 把学校课堂、家庭练习、错题回炉、技能地图连接成闭环。
- 让袁佳乐感到学习是有趣、可控、能进步的。
- 让家长不用当老师，只需按操作台执行。
- 通过王牌钓手世界观持续强化正向反馈。

## 核心对象

### 客户

袁佳乐，上海虹口区小学四年级学生。

### 家长角色

爸爸是“大副”，不是审判员。大副只负责：

1. 启动练习。
2. 勾选状态。
3. 问一句费曼问题。
4. 拍照提交。
5. 到点收工。

### 系统角色

系统是“专属家庭教师 + 教研 + 学管 + 讲义制作 + 复盘系统”的合体。

## 每日运行链路

```text
Step 1 课堂对齐
  爸爸上传或口述语数外今天讲到哪里。
  如果是音频，系统转写并抽取关键词。

Step 2 作业批改
  上传照片。
  生成 grading-result.csv。
  每题绑定知识点、答案、错因、费曼状态。

Step 3 掌握度更新
  错题 -> 待回炉。
  做对但讲不清 -> 做对但需讲清。
  做对且讲清 -> 理解通过。
  跨次连续正确 -> 稳定掌握 / 已毕业。

Step 4 次日决策
  红色：休整。
  黄色：1-2题轻量回炉。
  绿色：12-20分钟短练习。

Step 5 生成练习
  先生成 JSON spec。
  再生成 PDF、批改模板、家庭教师操作台。

Step 6 正向反馈
  更新鱼币、图鉴、海域、Boss徽章。
  告诉袁佳乐：今天努力推进了哪里。
```

## 练习设计原则

每份每日练习最多包含：

- 进门回测：1-3题。
- 主知识鱼：1条。
- 回炉题：昨日错题变式。
- Boss题：1个关键理解点。
- 费曼讲题：1题。
- 家校雷达：语数外课堂三句话。
- 语音采集：课堂三句话和 Boss 讲题。

禁止：

- 因为全对而加题。
- 同一天塞太多知识点。
- 把错题变成惩罚。
- 用长篇说明占用孩子注意力。
- 让家长临场发挥讲课。

## 技能地图机制

每个子级技能点必须经历：

1. `未识别`：还没测。
2. `入门验证`：1题，判断是否知道规则。
3. `基础操练`：2-3题，确认能稳定做。
4. `变式迁移`：1题，换题型或情境。
5. `费曼解释`：能讲清。
6. `间隔复查`：D+3 / D+7。
7. `已毕业`：连续正确且讲清。

每个子级技能点出 1-5 题，不一次性刷完。错误题第二天变换题型继续操练。

## 音频输入机制

用户可以上传课堂反馈音频或家长口述音频。系统处理顺序：

1. 转写。
2. 抽取语文、数学、英语关键词。
3. 判断是否为学校当天内容、作业要求、老师强调、测验通知。
4. 写入课堂同步日志。
5. planner 生成下一天练习时优先对齐。

音频不直接决定练习，必须与技能地图和掌握度合并判断。

## 王牌钓手游戏化

当前阶段只使用王牌钓手，不使用宝可梦。

映射：

- 知识点：知识鱼。
- 错题：逃脱鱼。
- 每日练习：出海。
- 主知识点：今日海域目标。
- 难题：Boss。
- 积分：Switch2鱼币。
- 技能图谱：知识鱼图鉴。
- 阶段测验：深海猎手大赛。

80分达成后，才设计宝可梦第二世界观，用于更高级的技能收集和进化。

## 成功定义

短期成功：

- 袁佳乐愿意开始。
- 每天 12-20 分钟能完成。
- 错题第二天能追回一部分。
- 家长不崩溃。

中期成功：

- 数学基础分明显稳住。
- 语文阅读答题有模板。
- 英语核心句型不再散。
- 字迹速度和稳定性改善。

长期成功：

- 袁佳乐相信自己可以进步。
- 学习不再只靠逼迫。
- 家庭形成温和、稳定、高反馈的学习仪式。
