## D 到 B 的快速提升模型

### 定义

这里的“快速”不是高压突击，而是以最短路径消灭基础漏分。

```text
D -> C：减少不会做和漏题
C -> B：减少会做但错、会算但讲不清
B 稳住：跨日复查和题型迁移
```

### D -> C：前 2-3 周

目标：把“不会启动”和“明显基础错”降下来。

策略：

- 每晚 12-15 分钟。
- 每天 1 个 Boss。
- 每天 1-3 条逃脱鱼。
- 课堂语音对齐学校内容。
- 数学优先：运算顺序、退位减法、单位换算、基础应用题。
- 语文优先：课内字词、阅读答题句式。
- 英语优先：课内单词、be 动词、现在进行时。

验收：

- 进门回测正确率连续 5 天达到 70%。
- 费曼语音至少 3 次 pass。
- 错题第二天追回率达到 50%。

### C -> B：第 4-8 周

目标：从“知道规则”到“稳定迁移”。

策略：

- 每个子技能点进入 1-5 题阶梯。
- 每两天加入一题多解或一题多说。
- 每 4 次练习做一次 6-8 题小测。
- 语文阅读开始用模板化答案。
- 英语从填空转向口头句子产出。

验收：

- 同一技能点连续正确 2 次。
- 费曼 pass 至少 1 次。
- D+3 复查正确。
- 周报显示待回炉技能减少。

### B 稳住：第 9-12 周

目标：稳住基础分，准备后续拔优。

策略：

- 每周一次综合小测。
- 每周一次“海域挑战”。
- 数学加入浅拓展。
- 语文加入主题阅读。
- 英语加入分级阅读。

验收：

- 已毕业技能不少于 P0 技能的 70%。
- 新错题主要来自变式或表达，不再是基础规则缺失。
- 袁佳乐能主动说出“今天抓哪条鱼”。

## 网上资料对项目的校正

上海“双减”和作业管理政策强调减负提质、作业育人价值、作业研究与监测；这支持我们继续坚持“短回合 + 高反馈 + 可监测”，而不是把 D 到 B 理解成刷题量。

上海全员导师制强调家校真诚互动、科学教育方法和亲子关系；这支持我们把爸爸定位为“大副”，不让家长变成审判者。

国家课标是拔优的边界：后续提高到 B 后，拔优资源必须先不超纲地扩展理解，再逐步引入浅拓展。
