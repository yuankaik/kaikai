The generated PDF does not copy Ace Angler screenshots, fish photos, UI, or original assets. It uses official game mechanisms as learning inspiration and creates original fish-shadow cards with ReportLab.

## Update protocol

After each submission:

1. Grade the paper.
2. Write errors to grading-log.csv.
3. Update mastery-state.csv.
4. Extract classroom briefs to classroom-brief-log.csv.
5. Regenerate the atlas when enough mastery status has changed.
