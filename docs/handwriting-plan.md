# 收杆练字方案 v1

## 目标

每天只练3个字，作为练习结束前的“收杆动作”，不把练字变成新负担。

## 字源

优先从当天语文课本/项目语文学习池抽取：

1. 当天古诗、成语、名言中的核心字。
2. 结构有训练价值的字：左右结构、上下结构、半包围、易错部件。
3. 不选过生僻、笔画过密、当天无语文关联的字。

当前字库：`records/handwriting-words.csv`。

## 页面结构

每个字一行：

- 1个示范格：行楷风格示范字，放在方格中。
- 5个练习格：空心方格，内部十字虚线。
- 每格建议 18mm 到 20mm，适合四年级孩子实际书写。

## 字体策略

优先使用系统行楷/楷体字体：

- 华文行楷 / STXingkai
- 楷体 / KaiTi
- Windows 缺字体时回退到 serif

原则：示范字接近行楷，但不追求成人书法味；孩子练习目标是“结构稳、收笔清、横平竖直中带一点连贯”。

## Day2 样张

《宿新市徐公店》：`宿 / 新 / 疏`

选择理由：

- `宿`：上下结构，宝盖头容易写散，适合练整体收紧。
- `新`：左右结构，左右比例容易失衡，是高频实用字。
- `疏`：课文诗句核心字，笔画较多，适合练慢写和部件位置。

## 嵌入规则

- 放在每日 PDF 最后一段，标题固定为“收杆练字”。
- 不计入错题惩罚，只记录完成情况。
- 字写得不好时不返工，只挑一个字说“这一竿最稳”。
- 当天黄色状态：只写每字2格。
- 当天红色状态：跳过练字。
