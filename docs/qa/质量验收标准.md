# 质量验收标准

生成日期：2026-06-04

## PDF 验收

每日 PDF 必须通过：

- A4 页面。
- 主体题目字号不低于 12pt。
- 正文说明字号不低于 10pt。
- 页脚和小标签不低于 8.5pt。
- 数字单位不可断行：`255ml`、`6升641ml`、`10cm`。
- 每页有明确标题、练习区、反馈区。
- 不出现过密小字。
- 不出现大面积空白页。
- Day3 及以后不在每日练习 PDF 中加入书写反馈页。
- 第 3 页必须提供课堂反馈/费曼语音提交说明、提交清单和孩子可见进度反馈。

## 题目验收

每题必须通过：

- 标准答案可验证。
- 题目难度符合当天状态。
- 绑定技能点。
- 绑定错因。
- 有错后变式策略。
- 不用含糊答案。

风险题必须标记：

- 非整除。
- 多答案。
- 题干不完整。
- 超出四年级范围。
- 需要图形但 PDF 没图。

## 教学验收

每天只允许：

- 1 条主知识鱼。
- 1 个 Boss 点。
- 1 次费曼讲题。
- 1 组课堂对齐。
- 1 个收工反馈。

禁止：

- 新知识过载。
- 错题惩罚化。
- 因全对追加题。
- PDF 文案比题目还多。

## 家长操作台验收

操作台必须能让家长 30 秒看懂：

- 今天是什么模式。
- 今天只盯哪三件事。
- 费曼问什么。
- 做到哪里收工。
- 拍照要拍哪些区域。

## 数据验收

每日生成后必须有：

- `practice/specs/DayX-*.json`
- `DayX-*.pdf`
- 可选 `submissions/DayX/grading-result.csv`
- 更新后的 `家庭教师操作台.md`

验证命令：

```powershell
python -m unittest discover -s tests -v
python scripts\check_pdf_quality.py --pdf "Day3-海王龙的逆向追踪.pdf"
```
