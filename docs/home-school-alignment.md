# 家校对齐执行规则

生成日期：2026-06-04

## 目标

让每日练习和学校课堂互相咬合：学校今天刚讲过的内容，晚上可以被轻量复现；我们今天练过的关键点，第二天或本周在课堂出现时，袁佳乐能产生“我见过”的熟悉感。

## 每日输入

每天练习纸面固定加入“家校雷达 - 今天课堂三句话”。袁佳乐只需要用简单语言说语数外各一句：

- 语文今天讲到什么？记得一个词、课文、古诗或老师强调点。
- 数学今天讲到什么？自己觉得会、半会、不会。
- 英语今天讲到什么？记得一个单词、句子或课文主题。
- 大副补充：老师强调、作业、测验、默写、复习范围。

## 执行边界

- 不追问细节，不现场加测。
- 孩子说不清时，记录“说不清”，这是有效信息。
- 家长只补学校客观信息，不把课堂回忆变成批评。
- 红色状态只记录，不追加练习。

## 系统处理

拍照后，系统把家校雷达录入 `records/classroom-brief-log.csv`，再写入 `records/school-sync-log.csv`。次日生成时按优先级处理：

1. 老师明确说明明天/本周要测：P0，必须进入次日练习。
2. 当天课堂新讲且袁佳乐说“不会/半会”：P0，安排 1 道轻量题。
3. 当天课堂新讲且他说“会”：P1，安排 1 个口头或进门测检查。
4. 与错题重合：P0，进入逃脱鱼回炉。
5. 与当前冲刺目标无关：只记录，不占纸面。

## 生成原则

每日练习的内容来源排序为：

1. 昨日逃脱鱼。
2. 今日/本周课堂同步。
3. 上海四年级下期末冲刺地图。
4. 个人薄弱知识点。
5. 游戏叙事包装。

游戏元素服务于家校对齐，不喧宾夺主。
