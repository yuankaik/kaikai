# 家长执行与正向反馈手册

生成日期：2026-06-04

## 大副定位

爸爸不是老师，不负责讲长课。爸爸是大副，只做五件事：

1. 开始时降低阻力。
2. 卡住时保护情绪。
3. 做对时给具体反馈。
4. 讲题时问一个为什么。
5. 到点收工。

## 每日三句话

开始：

> 船长，今天只抓昨天跑掉的几条鱼，抓到就收工。

卡住：

> 这条鱼先放回海里，我们换个鱼饵，不急。

收工：

> 今天完成就算赢，不加题。你已经把图鉴往前推了一点。

## 正向反馈规则

反馈必须具体，不夸空话。

| 情况 | 说法 |
|---|---|
| 做对 | 你这题先看运算顺序，这一步很稳。 |
| 做错 | 这条鱼跑掉了，说明它正好是系统要抓的薄弱点。 |
| 讲清 | 你刚才说“两次分不是一次分”，这就是 Boss 的弱点。 |
| 写字稳 | 这一格结构最稳，我们只圈这一格。 |
| 不想做 | 今天降级，只做一竿，做完就收工。 |

禁止：

- “你怎么又错了”
- “这么简单都不会”
- “别人都会”
- “做完再加几题”
- “不做就取消游戏/体育”

## 情绪状态

### 绿色

- 12-20 分钟。
- 可做进门回测、主练习、费曼，并提交课堂/讲题语音。
- 全对不加题。

### 黄色

- 5-10 分钟。
- 只做 1-2 题。
- 不讲新知识。
- 完成后马上收工。

### 红色

- 不做学科推进。
- 只保留关系动作：看图鉴、圈进度、选海域。
- 记录状态，第二天再说。

## 费曼讲题

爸爸只问：

> 船长，我不太明白，为什么这题不能这样做？

判断：

- `pass`：能说步骤和原因。
- `weak`：会算，但说不清为什么。
- `fail`：沉默、乱讲、只背口诀。

注意：

- 讲不清不是失败，是系统发现薄弱点。
- 不要求术语准确，要求意思对。
- 爸爸不要抢着讲答案。

## 课堂音频对齐

用户可以上传课堂反馈音频或口述音频。家长只需要说：

- 语文今天讲了什么。
- 数学今天讲了什么。
- 英语今天讲了什么。
- 老师有没有强调默写、测验、复习范围。

系统负责转写、抽关键词、对齐技能地图。

## 袁佳乐反馈看板

每天给孩子看的反馈只保留三项：

1. 今日抓到的鱼。
2. 今日追回的逃脱鱼。
3. 下一个可解锁目标。

示例：

```text
今日渔获：
- 追回：44-18 退位鱼
- Boss发现：除数不能合并
- 下一条鱼：容量单位换算

图鉴进度：2/93 -> 3/93
Switch2鱼币：1835 -> 1870
```

## 长期心态

我们不承诺短期奇迹。我们承诺：

- 每天发现一个真实问题。
- 每天追回一点点。
- 每天让袁佳乐知道努力有位置。
- 每天让家长知道该做什么、不该做什么。
