# 王牌钓手学习化设计笔记

生成日期：2026-06-04

## 游戏元素映射

- Marine Medal Mania / 水族馆主题乐园 -> 每日出海任务。
- Medals / 奖章 -> Switch2鱼币积分。
- Aquarium / 鱼类图鉴 -> 知识鱼图鉴。
- Normal Rod / Super Rod / Monster Rod -> 口算热身 / 错题回炉 / Boss题。
- Monster Class fish -> 当天核心难点。Day2为巨齿鲨Boss。
- Fish Gacha Machine -> 不直接照搬随机抽奖，改为完成后解锁图鉴，避免学习目标被随机化。
- Shark Fever -> 阶段挑战赛元素，适合周测，不塞进每日基础练习。

## Day2生成原则

Day2保留3页：两页练习 + 一页收杆练字。页眉固定包含海域、Switch2进度条、知识鱼图鉴、状态勾选。
题量保持轻，游戏元素服务于任务结构，不作为额外阅读负担。
