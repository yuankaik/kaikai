# 上海小学课程与教材来源说明

生成日期：2026-06-04

## 使用原则

本项目把资料分成三层，避免“教材目录”“孩子掌握情况”“每天生成策略”混在一起。

- `records/shanghai-primary-knowledge-map.csv`：上海小学 1-5 年级主知识地图，用来回答“学校体系里有哪些知识鱼”。
- `records/knowledge-matrix.csv`：袁佳乐个人掌握矩阵，用来回答“这些知识鱼他现在抓得怎么样”。
- `records/mastery-state.csv`：每次批改后的动态状态，用来回答“明天要回炉哪几条逃脱鱼”。

## 已采用来源

1. 上海市教育委员会《2025 年秋季上海市编写教学用书目录》。用于确认上海小学语文、数学、英语教材体系，特别是语文五四学制、小学数学试用本、英语牛津上海版。
2. 上海市人民政府/上海市教育委员会《2025 学年度上海市普通中小学课程计划》。用于确认小学 1-5 年级课程结构。
3. 上海世纪外语资料对《英语（牛津上海版）》的教材背景说明。用于确认牛津上海版英语教材属性。
4. 沪教版小学数学目录资料。用于补足数学分册目录的结构参考。该类资料只作为目录校对参考，不作为唯一权威来源。
5. 义务教育课程方案和课程标准 2022 年版。用于建立语文、数学、英语的能力域框架，但本项目的日常练习仍以上海教材目录和袁佳乐错题为主。

## 当前结论

- 上海小学阶段按 1-5 年级组织，项目知识地图必须覆盖 G1-G5。
- 四年级下期末冲刺不能只看“当前一册”，还要回收 G1-G4 的基础知识漏洞。
- 每日练习生成顺序必须是：当天状态 + 批改错题 + 上海知识地图 + 个人掌握矩阵 + 游戏叙事模板 + 统一 PDF 模板。

## 风险说明

网上能找到的分册目录资料来源质量不一。正式进入长期服务前，需要逐步用学校实际课本、老师发的复习资料、单元卷和袁佳乐作业照片校正这张知识地图。
