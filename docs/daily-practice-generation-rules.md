# 每日练习生成规则 v4

## 总原则

每日练习不提前批量生成。每次只生成“下一天”一份正式 PDF。

正式 PDF 必须在生成时直接包含以下模块：

1. 海域页眉：Switch2 进度条、知识鱼图鉴、今日海域、状态勾选。
2. 进门测：昨日逃脱鱼或关键基础点。
3. 当日主练习：当天只抓 1-2 条主知识鱼。
4. 家校雷达：语文、数学、英语课堂三句话。
5. 费曼讲题：爸爸现场勾选 pass / weak / fail。
6. 今日战果：上钩数、逃脱鱼、鱼币、状态。
7. 收杆练字：每日 3 个示范字。

`家校雷达` 和 `收杆练字` 都不是事后追加内容，而是每日练习生成器的一部分。

## 家校雷达模块

模块目标：让每日练习与学校课堂同步。

纸面字段：

- 语文今天讲到：课文、词语、古诗、老师强调点。
- 数学今天讲到：知识点或题型，并让孩子勾选会/半会/不会。
- 英语今天讲到：单词、句型、课文主题。
- 大副补充：作业、测验、默写、老师复习范围。

执行原则：

- 袁佳乐只说关键词，不考试。
- 说不清也是有效信息，记录为“unknown”。
- 红色状态只记录，不追加练习。
- 次日生成时，课堂同步内容优先级高于普通新题。

数据入口：

- `records/classroom-brief-log.csv`
- `records/school-sync-log.csv`
- `docs/home-school-alignment.md`

## 收杆练字模块

模块实现：`scripts/handwriting_module.py`

字库：`records/handwriting-words.csv`

标准版：

- 每日 3 个字。
- 每字 1 个示范格。
- 每字 5 个练习格。
- 方格内十字虚线。
- 示范字优先行楷，回退楷体。

降级版：

- 黄色状态：每字只写 2 格。
- 红色状态：跳过练字。

## 次日选题优先级

1. 昨日逃脱鱼。
2. 今日/本周课堂同步内容。
3. 上海四年级下期末冲刺地图。
4. 袁佳乐个人薄弱知识点。
5. 游戏叙事包装。

先批改，再生成次日练习。禁止提前批量生成后续每日练习。禁止在成品 PDF 上补丁式追加核心模块。
