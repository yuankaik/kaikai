# 掌握度规则 v1

目标：让“会了没有”从感觉判断变成可追踪状态。

## 状态定义

- `待回炉`：本次做错，或讲不清关键步骤。
- `做对但需讲清`：答案正确，但费曼讲题未通过或未验证。
- `理解通过`：答案正确，且能用自己的话讲清楚。
- `稳定掌握`：跨次练习连续正确至少2次。
- `已毕业`：连续正确至少3次，且至少1次费曼通过。

## 错因分类

- `知识点缺失`：不知道规则或公式。
- `题意理解`：读题关系搞反、条件漏看。
- `计算错误`：会方法但算错。
- `格式问题`：答案、单位、步骤不规范。
- `粗心`：看错数字、抄错、漏题。

## 次日安排规则

- 有 `知识点缺失` 或 `题意理解`：次日必须回炉，不直接加难度。
- 只有 `计算错误` 或 `粗心`：次日可继续推进，但加入1道同型检查题。
- 费曼未通过：即使做对，也不能毕业。
- 当天情绪为红色：不生成新练习，只安排轻量复盘或休整。
- 当天情绪为黄色：最多安排进门测 + 1道回炉题 + 1个收工奖励。

## 项目铁律

先批改，再生成次日练习。禁止提前批量生成后续每日练习。
