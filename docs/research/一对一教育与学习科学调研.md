# 一对一教育与学习科学调研

生成日期：2026-06-04

## 调研目的

把线下高端一对一辅导机构的服务结构拆解为可执行产品机制，并结合学习科学和青少年心理特点，重构袁佳乐专属家庭教师系统。

本系统不复制任何机构课程，而是抽象其服务能力：测评、计划、专属讲义、过程跟踪、阶段反馈、家校同步。

## 机构方法论拆解

### 新东方：多对一服务链路

新东方个性化学习中心页面强调：一对一不仅是提分，也包括学习习惯和学习方法；其服务特色包括咨询顾问剖析问题、教研团队定制课程、授课教师针对性授课、学管跟踪进度、阶段测试、月度反馈。页面服务流程包含水平测试、评估报告、授课计划、教学跟踪、阶段评估测试。

产品转译：

- `诊断层`：每天从作业照片、课堂音频、家长观察中做微测评。
- `教研层`：技能地图与题型模板决定练什么，不靠临时灵感。
- `授课层`：每日 PDF 是孩子可执行的短讲义。
- `学管层`：家庭教师操作台给家长三件事。
- `反馈层`：掌握度、鱼币、知识图鉴、错题回炉组成闭环。

来源：https://qd.xdf.cn/zhuanti/gxh.html

### 学而思 / 好未来：数据驱动与个性化路径

好未来官网公开强调“爱和科技推动终身成长”，并在学习设备与 AI 学习内容中提到基于学生学习数据进行精准分析、提供个性化学习路径和内容。公开资料还强调教育科技、智能教育、内容解决方案与学习服务。

产品转译：

- 练习不是静态卷子，而是由数据驱动。
- 每题必须带 `knowledge_id`、难度、错因、下一步动作。
- 每天只推进孩子当前最需要的 1-2 个技能点。
- 通过图鉴进度让孩子看到“我已经解锁了什么”。

来源：https://www.100tal.com/

### 精锐教育：学习力与个性化环境

精锐教育公开材料和第三方资料反复出现“学习力”“个性化教育”“一对一”“主动学习”“多元互动”等关键词。其方法论可抽象为：个性化诊断、专属学习环境、学习力培养、从被动听课转向主动学习。

产品转译：

- 袁佳乐不是只刷题，而是训练学习力：读题、讲题、复盘、迁移。
- 每日练习加入费曼讲题，让孩子从“会算”走向“会解释”。
- 家庭环境不制造压迫感，只做短回合、高反馈。

来源示例：https://edu.sina.com.cn/zxx/2014-05-28/1132420579.shtml

## 学习科学原则

### 间隔重复 / 艾宾浩斯方向

间隔练习优于临考前集中突击。The Learning Scientists 对 spaced practice 的说明是：把同样学习时间分散到更长时间，会带来更持久学习；学习需要提前计划，并在后续回到旧知识保持新鲜。

产品规则：

- 错题进入 `D+1 / D+3 / D+7 / D+14` 复习池。
- 第二天必须变式回炉，不原题机械重复。
- 已毕业技能仍进入低频巡航复查。

来源：https://www.learningscientists.org/blog/2016/7/21-1

### 检索练习

检索练习要求学生从记忆中提取已学内容，而不是马上看答案或笔记。The Learning Scientists 指出，检索后要对照材料检查，获得知道什么与不知道什么的反馈。

产品规则：

- 每日开头是“进门回测”，不翻昨天 PDF。
- 回测题少而关键，验证昨天薄弱点是否还在。
- 批改后立刻更新掌握度。

来源：https://www.learningscientists.org/blog/2016/6/23-1

### 费曼学习法

Kansas State University 的学习中心把费曼法拆成 Study、Teach、Fill the gaps、Simplify 四步；核心是通过教给别人暴露不清楚的地方，再补缺、简化。

产品规则：

- 做对但讲不清，不能毕业。
- 爸爸只问一个问题：为什么？
- 讲题通过标准写入 `feynman_status`。
- 费曼失败不是批评，而是下一次换鱼饵。

来源：https://www.k-state.edu/aac/success-tools/collection/feynman_technique.html

## 青少年心理与家庭执行

袁佳乐现在是四年级，既有儿童阶段对游戏化、即时反馈的依赖，也会逐步进入更强自尊和自主需求阶段。CDC 对 4-11 岁儿童资料强调，这一阶段习惯与行为会影响长期健康；对青少年资料强调，12-19 岁是影响当前和未来健康的关键期。

产品规则：

- 保护自尊优先于当天进度。
- 红色状态不学新内容。
- 黄色状态只做 1-2 题。
- 全对不加题，避免把成功变成惩罚。
- 错题叫“逃脱鱼”，不叫粗心、不贴负面标签。

来源：

- https://www.cdc.gov/parents/children/index.html
- https://www.cdc.gov/parents/teens/index.html

## 家校对齐

CDC 关于 school connectedness 的资料强调，学生感到学校成人和同伴关心自己和学习时，更可能有更高成绩、出勤与积极健康行为。家庭系统不能脱离学校课堂，必须知道今天学校语数外讲到哪里。

产品规则：

- 每日收集语文、数学、英语课堂三句话。
- 用户可上传音频，系统转写后抽取关键词。
- 练习生成优先对齐当天课堂，其次才是长期技能地图。
- 家长只补“学校今天讲了什么”和“孩子是否会/半会/不会”。

来源：https://www.cdc.gov/youth-behavior/school-connectedness/index.html

## 对袁佳乐的产品结论

1. 新东方式服务链路给我们项目结构：诊断、计划、讲义、跟踪、反馈。
2. 学而思/好未来式数据路径给我们算法结构：每题可追踪、每次有状态、每轮有路径。
3. 精锐式学习力理念给我们教学目标：不是只提分，而是让孩子主动解释、迁移、复盘。
4. 艾宾浩斯和检索练习给我们复习机制：错题第二天变式，之后间隔巡航。
5. 费曼法给我们毕业标准：会做不够，讲清才算理解。
6. 青少年心理给我们边界：保护关系、保护自尊、短回合、即时正反馈。
7. 王牌钓手给我们动机外壳：知识鱼、Boss、图鉴、鱼币、海域。
