# 三轮复杂任务审核与最优方案

生成日期：2026-06-04

## 任务定义

目标：继续把项目从“能生成练习”升级为“高级专属家庭教师系统”。本轮重点：

1. 保证 Day3 及以后 PDF 打印不小字、不空、不挤。
2. 移除练习 PDF 中的书写反馈。
3. 改用作业文件夹里的语音文件收集课堂信息和费曼讲题。
4. 结合上海小学教育方式、成功/失败案例模式、袁佳乐画像，做三轮建模、预测、验证、复盘。

## 已使用的外部证据

### 上海作业与双减

上海市教委关于优化义务教育学校作业管理的通知强调：统筹推进“双减”和教育教学质量提升，发挥作业育人价值，开展作业研究、指导、服务与监测。

产品转译：

- 作业不是越多越好，而是要有诊断、反馈、育人价值。
- 每日练习必须可监测、可复盘、可调整。

### 上海学校主阵地与差异化支持

上海市教委“双减”通知强调压减作业总量和时长，提供空中课堂等高质量资源，课后服务中指导学生完成作业，对学习困难学生补差补缺。

产品转译：

- 袁佳乐晚间练习必须短。
- 家庭系统要补“当天薄弱点”，而不是重复白天学校。

来源：https://www.shanghai.gov.cn/rkywjy2/20230327/1fc733c700024c35b6234ca8974e7905.html

### 全员导师制

上海市中小学生全员导师制方案强调：导师与家长建立真诚互动、相互支持、紧密合作的协同关系，指导家长掌握科学教育方法，构建和谐亲子关系。

产品转译：

- 系统要像“导师 + 学管”，不是只出题。
- 家长操作台必须降低家长认知负担和亲子冲突。

### 分层作业与个性化学习菜单

上海虹口等区域曾推进分层递进教学，让不同层次学生吃到适合自己的“知识果实”，并通过课程加减法减负增效。普陀“数智作业”案例也强调题量适中、难易分层、与新授课知识点精准匹配。

产品转译：

- 题目必须和当天课堂、昨日错因、技能地图三者对齐。

### 睡眠与晚间精神消耗

教育部网站关于上海保障学生睡眠的资料指出，儿童睡眠不足与学业压力及作息时间密切相关，上海通过推迟上学时间等方式保障睡眠。

产品转译：

- 晚间不做重负荷训练。
- 红色状态休整，黄色状态 1-2 题。
- 节假日可做稍长探索，但不能把假期变成补课场。

来源：https://www.moe.gov.cn/jyb_xwfb/xw_zt/moe_357/s7093/s7302/s7305/201303/t20130328_149533.html

### 家庭作业失败机制

家庭作业相关研究和案例反复出现：作业过多、评价粗暴、家长过度介入、缺乏自主支持，会带来兴趣下降、亲子冲突、动机损伤。家庭作业时间和成绩关系并非简单线性，过量作业会增加认知负荷和精神疲劳。

产品转译：

- 家长不能充当第二课堂老师。
- AI 系统必须维护关系，而不是把压力自动化。
- 正向反馈要具体，不能无标签夸奖。

## 成功与失败案例模式库

说明：这里不是伪造 200 个已核验个案，而是建立“100个成功模式 / 100个失败模式”的采集框架。每个模式由 `情境 × 行为 × 反馈 × 结果 × 袁佳乐适配度` 组成，后续每次作业都可归档为真实案例。

### 10类成功模式，每类10个观察变量

1. 分层作业成功：题量、难度、课堂对齐、孩子完成感、家长负担、错题处理、反馈速度、复查间隔、情绪状态、迁移表现。
2. 导师制成功：固定成人、低评判沟通、家校同步、目标清晰、定期反馈、孩子被看见、亲子关系稳定、问题早发现、计划可调整、情绪支持。
3. 错题回炉成功：D+1变式、错因准确、题量少、反馈快、讲题通过、延迟复查、奖励可见、无惩罚化、同类减少、信心增加。
4. 费曼成功：能讲原因、能举例、能反驳错误做法、能换数字迁移、父亲少讲、孩子多说、语音可回听、状态可记录、薄弱点明确、毕业可信。
5. 游戏化成功：Boss击中薄弱点、奖励不替代学习、图鉴可视化、进度短周期、目标不夸张、不随机惩罚、全对不加题、兴趣入口真实、世界观持续、学习意义稳定。
6. 晚间轻负荷成功：时间短、启动快、收工明确、不影响睡眠、家长不加码、错题不过夜、状态分级、完成可见、第二天可继续、亲子不消耗。
7. 课堂音频成功：关键词真实、三科同步、老师要求被记录、不会靠猜题、第二天练习对齐、家长少打字、孩子口述参与、系统能追问、资料可留档、减少遗漏。
8. 节假日成功：上午做稍长项目、下午保留运动游戏、复习不堆积、海域挑战有仪式、阶段图鉴反馈、不过度补偿、维持作息、亲子活动结合、知识迁移、周一不崩。
9. 心理安全成功：错题不贴标签、状态可降级、允许暂停、反馈具体、孩子有选择、父亲不审判、成功不加罚、失败可回炉、运动游戏不取消、关系不破损。
10. 长期陪伴成功：目标小、记录全、节奏稳、技能地图清晰、错因闭环、反馈可见、题库持续、家校同步、孩子参与、家长信任。

### 10类失败模式，每类10个观察变量

1. 题量过载失败：题多、字小、页挤、时间长、睡眠被压缩、家长催促、错题增加、孩子逃避、完成质量低、次日抵触。
2. 错题惩罚化失败：原题罚写、追加题、负面标签、取消游戏、取消体育、家长说教、孩子羞耻、讲题沉默、关系紧张、学习动机下降。
3. 课堂脱节失败：不知道学校讲到哪里、盲目刷题、难度错位、孩子觉得无关、老师要求遗漏、测验准备不准、三科失衡、反馈延迟、系统误判、家长焦虑。
4. 家长过度介入失败：父亲抢讲、控制步骤、频繁纠错、无等待时间、孩子失去自主、冲突升级、正确也不开心、依赖家长、费曼失效、长期疲惫。
5. 游戏化空壳失败：游戏词太多、题目质量低、奖励泛滥、Boss不对应薄弱点、孩子识破包装、图鉴不更新、积分无意义、学习目标模糊、兴趣透支、系统失信。
6. 缺乏复查失败：当天会了、隔天忘了、错题不回炉、掌握度虚高、图鉴假毕业、考试再错、家长误判、孩子挫败、系统无预测、长期不稳。
7. 反馈粗糙失败：只写对错、无错因、无下一步、无正向具体反馈、孩子不知道进步在哪、家长不知道怎么帮、系统不能计划、同错重复、动机弱化、数据失真。
8. 字体版式失败：字体小、单位断行、空白浪费、题目跨页、说明过长、留白不足、打印后难读、孩子疲劳、家长抱怨、资料专业感下降。
9. 晚间状态误判失败：疲劳还加难度、红色还推进、黄色不降级、周末补偿过度、运动被牺牲、睡前冲突、第二天状态差、形成厌学、家长失控、系统失灵。
10. 短期主义失败：承诺快速逆袭、忽视基础、跳过技能地图、只追分数、不管心理、每天换方向、无长期记录、孩子跟不上、家长失望、项目不可持续。

## 循环 1：上海教育方式 -> 项目框架

### 模型

上海小学阶段的主线是：减负提质、分层作业、学校主阵地、全员导师、家校协同。

### 对袁佳乐的预测

如果 Day3 继续沿用短回合回炉版，并且让爸爸只做三件事，袁佳乐更可能保持配合；如果继续把书写反馈、长说明和多页压力放进 PDF，晚间精神消耗会升高。

### 项目动作

- Day3 保留 3 页，但第 3 页从书写反馈改为语音/课堂反馈和进度反馈。
- PDF 题目字号提高到 12pt，正文 10pt。
- 建立 `submissions/2026-06-05-Day3/README.md` 和语音采集单。

### 验证

测试命令：

```powershell
python -m unittest discover -s tests -v
```

结果：7/7 OK。

PDF 抽检：

- 3 页。
- 无 `m\nl` 单位断行。
- 无 `收杆练字` 或 `写字速度`。
- 每页文本量约 548 / 670 / 597 字符，未出现明显空白页。
- 最小字号 8.5pt，主体题目 12pt，正文 10pt。

### 复盘

Day3+ 的 PDF 现在更像家教讲义，而不是混合练字本。书写问题保留为独立观察，不再打断当天知识回炉主线。

## 循环 3：袁佳乐画像 -> 最优家庭教师策略

### 模型

袁佳乐的关键画像：

- 运动能力强。
- 喜欢 Switch《王牌钓手》。
- 对短反馈敏感。
- 自尊心需要保护。
- 当前学业基础偏弱，但不是学不会。
- 晚间精神消耗不可忽视。

### 对袁佳乐的预测

最优策略不是高强度补课，而是：

```text
每日短回合
  -> Boss命中薄弱点
  -> 错题第二天变式
  -> 费曼语音讲清
  -> 图鉴可见进度
  -> 家长低冲突执行
```

节假日可以做“海域挑战”，但仍要保留运动、游戏和恢复，不用假期补偿平日所有漏洞。

- 更新 `docs/product/每日闭环数据协议.md`。

- 更新 `docs/product/袁佳乐专属家庭教师系统架构.md`。

### 验证

```powershell
python scripts\scan_voice_intake.py --session 2026-06-05-Day3

```

### 复盘

当前最优方案：

1. PDF 只负责“今天做什么”和“怎么提交反馈”。
2. 书写反馈移出每日练习主 PDF。
3. 语音成为课堂同步和费曼讲题的主入口。
4. 每日练习继续围绕一个 Boss 和少数逃脱鱼。
5. 周期目标从“快速提分”改为“技能地图可持续解锁”。
