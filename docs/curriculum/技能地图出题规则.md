# 技能地图出题规则

生成日期：2026-06-04

## 总原则

技能地图是系统的骨架。每日练习只是技能地图的一个小切片。

每个子级技能点按 1-5 题设计，从入门到扩展，不一次性刷爆。

## 难度阶梯

| 阶段 | 题数 | 目标 |
|---|---:|---|
| 入门 | 1 | 知不知道规则 |
| 基础 | 1-2 | 能不能稳定做 |
| 易错 | 1 | 会不会踩常见坑 |
| 变式 | 1 | 换情境还能不能迁移 |
| 费曼 | 1个讲题 | 能不能讲清为什么 |

## 出题协议

每道题必须绑定：

- 科目。
- 年级学期。
- 一级知识点。
- 子级技能点。
- 难度。
- 标准答案。
- 常见错因。
- 错后第二天变式策略。
- 做对后的延迟复查策略。

## 错题回炉

错题第二天必须变式，不直接原题罚写。

变式方式：

- 换数字。
- 换语境。
- 换问法。
- 加一步解释。
- 从选择题变成判断题。
- 从计算题变成讲题。

例：

```text
原错题：
480÷4 + 480÷6 被写成 480÷(4+6)

第二天变式：
300÷5 + 300÷10 = 300÷(5+10) 对吗？

费曼问题：
为什么两个除法不能把下面的数加起来？
```

## 三科规则

### 数学

优先 P0：

- 运算顺序。
- 除法陷阱。
- 单位换算。
- 有余数除法。
- 退位减法。
- 倍数关系。
- 基础应用题。

数学每题必须能验算。

### 语文

优先 P0：

- 课内字词。
- 看拼音写词语。
- 段意概括。
- 中心句。
- 阅读答题句式。
- 修辞判断。

语文重点不是多刷，而是答案结构模板。

### 英语

优先 P0：

- 课内单词。
- be 动词。
- 现在进行时。
- there be。
- 问答句。
- 大小写和标点。

英语必须对齐当天课本内容和老师要求。

## 毕业规则

一个子级技能点毕业必须满足：

- 至少 3 次正确。
- 至少 1 次费曼 pass。
- 至少 1 次延迟复查正确。

毕业后不消失，进入低频巡航。
