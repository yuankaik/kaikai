# 上海小学数学 - 六年级 月考
## 百分数（一）

**时间：45分钟  满分：100分**

### 一、填空（每空2分，共20分）
1. 45%读作(    )。
2. 0.35用百分数表示是(    )。
3. 3/4用百分数表示是(    )。
4. 80%化成分数是(    )。
5. 六成=(    )%  三成五=(    )%
6. 产品合格率=(    )÷(    )×100%

### 二、判断（每题2分，共10分）
1. 百分数就是分母是100的分数。(    )
2. 一个数的80%一定小于这个数。(    )
3. 出勤率最高是100%。(    )

### 三、选择（每题3分，共15分）
1. 下面能用百分数表示的是(    )。
   A.一根绳长3/5米  B.男生占全班的3/5  C.苹果重3/5千克
2. 把25%的百分号去掉，这个数(    )。
   A.扩大100倍  B.缩小100倍  C.不变

### 四、计算（共20分）
1. 60的35%是多少？（5分）
2. ( )的40%是24？（5分）
3. 28是40的百分之几？（5分）
4. 比50多20%的数是多少？（5分）

### 五、解决问题（共35分）
1. 班里有50人，今天出勤48人，出勤率是多少？（8分）
2. 一本书原价40元，打八折出售，现价多少？（8分）
3. 油菜籽出油率38%，500kg油菜籽能榨多少油？（9分）
4. 商场促销满200减50，实际相当于打几折？（10分）

三、1.B 2.A
四、1.21 2.60 3.70% 4.60
五、1.96% 2.32元 3.190kg 4.七五折
