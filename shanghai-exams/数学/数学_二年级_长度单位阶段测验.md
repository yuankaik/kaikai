# 上海小学数学 - 二年级 阶段测验
## 长度单位

**时间：30分钟  满分：100分**

### 一、填空（每空2分，共20分）
1. 量比较短的物体用(    )作单位。
2. 量比较长的物体用(    )作单位。
3. 1米=(    )厘米
4. 200厘米=(    )米
5. 手掌宽约6(    )
6. 教室长约8(    )
7. 铅笔长约15(    )

### 二、在○里填＞＜＝（每题2分，共12分）
1. 1米○99厘米  2. 50厘米○5米  3. 200厘米○2米
4. 3米○30厘米  5. 80厘米○8米  6. 1米20厘米○120厘米

### 三、量一量画一画（共18分）
1. 画一条5厘米长的线段。（6分）
2. 画一条比3厘米长2厘米的线段。（6分）
3. 量出下面线段几厘米。（6分）

### 四、计算（每题4分，共20分）
1. 3米+5米=(    )米
2. 46厘米-20厘米=(    )厘米
3. 1米-40厘米=(    )厘米
4. 30厘米+70厘米=(    )米
5. 2米50厘米+1米50厘米=(    )米

### 五、解决问题（共30分）
1. 一根绳子长8米，剪去3米，还剩多少米？（6分）
2. 小明身高1米25厘米，爸爸身高1米78厘米，爸爸比小明高多少？（8分）
3. 学校走廊长30米，每隔5米放一盆花，一共放几盆？（8分）
4. 一根彩带长2米，做花用去80厘米，还剩多少厘米？（8分）

## 参考答案
一、1.厘米 2.米 3.100 4.2 5.厘米 6.米 7.厘米
二、1.＞ 2.＜ 3.＝ 4.＞ 5.＜ 6.＝
三、（略）四、1.8 2.26 3.60 4.1 5.4
五、1.5米 2.53厘米 3.7盆 4.120厘米
