## 六、解决问题（每题5分，共25分）

1. 学校买了5个篮球和8个足球，共花了824元。每个篮球68元，每个足球多少元？（列方程解答）

2. 一块平行四边形的地，底长120米，高80米。如果每平方米收小麦0.75千克，这块地共收小麦多少千克？

3. 一个梯形果园，上底80米，下底120米，高50米。如果每棵果树占地10平方米，这个果园可以种多少棵果树？

4. 把一张长24厘米、宽18厘米的长方形纸剪成同样大小的正方形，且没有剩余。正方形的边长最大是多少厘米？可以剪成多少个？

5. 甲、乙两辆汽车同时从相距480千米的两地相对开出，经过3小时相遇。甲车每小时行85千米，乙车每小时行多少千米？

## 参考答案

### 一、填空题

2. 36、4

4. 40、20
5. 1,2,3,4,6,8,12,24；2,3；4,6,8,12,24

10. 0.424（最大）、0.42（最小）

### 二、判断题
1. √  2. ×（需完全一样）  3. ×（一个数的最小倍数是它本身）  4. ×（需不为0）  5. ×

### 三、选择题
1. C  2. C  3. B  4. C（分母只含质因数2和5）  5. B

### 四、计算题
1. （1）0.3 （2）6 （3）1 （4）40 （5）10 （6）4 （7）0.09 （8）9
2. （1）3x=18，x=6
   （2）x-3=7，x=10
   （3）2.5x=6.5，x=2.6
3. （1）3.6×(99+1)=3.6×100=360
   （2）12.5×8×0.4×0.25 = (12.5×8)×(0.4×0.25) = 100×0.1 = 10

### 六、解决问题
1. 设每个足球x元。5×68+8x=824，340+8x=824，8x=484，x=60.5  答：每个足球60.5元。
2. 120×80=9600平方米，9600×0.75=7200千克  答：共收小麦7200千克。
3. (80+120)×50÷2=5000平方米，5000÷10=500棵  答：可以种500棵。
4. 24和18的最大公因数是6。边长最大6厘米。(24÷6)×(18÷6)=4×3=12个  答：可剪12个。
5. 480÷3-85=160-85=75千米/时  答：乙车每小时行75千米。
