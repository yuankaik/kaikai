# 上海小学数学 - 三年级 第一单元测验
## 时、分、秒

**时间：40分钟  满分：100分**

### 一、填空（每空2分，共20分）
1. 钟面上有(    )个大格，每个大格有(    )个小格。
2. 时针走1大格是(    )小时，分针走1小格是(    )分钟。
3. 1小时=(    )分钟，1分钟=(    )秒。
4. 秒针从12走到6，走了(    )秒。
5. 分针从3走到9，走了(    )分钟。

### 二、时间换算（每空2分，共16分）
1. 2小时=(    )分钟    2. 180秒=(    )分钟
3. 1小时30分=(    )分   4. 90分钟=(    )小时(    )分钟
5. 3分钟=(    )秒       6. 240分=(    )小时

### 三、看钟表写时间（每空3分，共15分）
1. 时针在8和9之间，分针指向6 → (    )时(    )分
2. 时针在3，分针指向12 → (    )时
3. 时针在11和12之间，分针指向3 → (    )时(    )分
4. 时针在1和2之间，分针指向9 → (    )时(    )分

### 四、计算经过时间（每空3分，共24分）
1. 8:30到9:15，经过了(    )分钟。
2. 电影2:10开始，3:40结束，放映了(    )小时(    )分钟。
3. 小明7:45到校，11:30放学，在校(    )小时(    )分钟。
4. 一节课从10:00开始，上了40分钟，(    )时(    )分下课。

### 五、解决问题（共25分）
1. 小红做作业用了35分钟，从晚上7:15开始，几点结束？（8分）
2. 上海到杭州的高铁需要45分钟，如果8:30发车，几点到达？（8分）
3. 小明每天睡眠9小时，如果他晚上9:00睡觉，第二天几点起床？（9分）

二、1.120 2.3 3.90 4.1,30 5.180 6.4
三、1.8,30 2.3 3.11,15 4.1,45
四、1.45 2.1,30 3.3,45 4.10,40
五、1.7:50 2.9:15 3.6:00
