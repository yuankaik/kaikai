# 上海小学数学 - 三年级 单元测验
## 面积

**时间：40分钟  满分：100分**

### 一、填空（每空2分，共20分）
1. 边长1厘米的正方形，面积是1(    )。
2. 边长1米的正方形，面积是1(    )。
3. 一个长方形长8cm，宽5cm，面积是(    )cm²。
4. 一个正方形边长6m，面积是(    )m²。
5. 600平方厘米=(    )平方分米
6. 4平方米=(    )平方分米
7. 300平方分米=(    )平方米

### 二、判断（每题2分，共10分）
1. 面积相等的两个长方形，形状一定相同。(    )
2. 边长4cm的正方形，周长和面积相等。(    )
3. 用8个1cm²的小正方形拼成任意图形，面积都是8cm²。(    )

### 三、选择（每题3分，共15分）
1. 一本数学书的封面面积约是3(    )。 A.cm² B.dm² C.m²
2. 教室的面积约是60(    )。 A.cm² B.dm² C.m²
3. 用12个1cm²拼长方形，有(    )种不同拼法。 A.2 B.3 C.4

### 四、计算面积（每题5分，共15分）
1. 长方形：长12cm，宽7cm
2. 正方形：边长9m
3. 长15m，宽8m的长方形

### 五、解决问题（共40分）
1. 一块长方形菜地长20m，宽15m，面积是多少？（8分）
2. 一面墙长6m，高3m，墙上有一扇2m²的窗户，实际粉刷面积？（10分）
3. 用长24米的篱笆围一个正方形鸡舍，面积是多少？（10分）
4. 客厅长8m宽6m，用边长2dm的方砖铺地，需要多少块？（12分）

## 参考答案
一、1.平方厘米 2.平方米 3.40 4.36 5.6 6.400 7.3
二、1.× 2.× 3.√
三、1.B 2.C 3.B
四、1.84cm² 2.81m² 3.120m²
五、1.300m² 2.16m² 3.36m² 4.40×30=1200块
