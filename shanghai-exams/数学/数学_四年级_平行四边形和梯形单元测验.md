# 上海小学数学 - 四年级 单元测验
## 平行四边形和梯形

**时间：40分钟  满分：100分**

### 一、填空（每空2分，共20分）
1. 两组对边分别(    )的四边形叫做平行四边形。
2. 只有一组对边平行的四边形叫做(    )。
3. 平行四边形有(    )条高。
4. 梯形的(    )叫做梯形的底。
5. 从平行四边形一条边上的一点到对边引一条(    )，这个点和垂足之间的线段叫做平行四边形的(    )。

### 二、判断（每题2分，共10分）
1. 平行四边形是特殊的长方形。(    )
2. 梯形只有一条高。(    )
3. 长方形和正方形都是特殊的平行四边形。(    )
4. 两个完全一样的梯形可以拼成一个平行四边形。(    )
5. 平行四边形容易变形。(    )

### 三、选择（每题3分，共15分）
1. 下面图形中，(    )是平行四边形。
2. 平行四边形有(    )条对称轴。 A.0 B.2 C.4
3. 等腰梯形有(    )条对称轴。 A.0 B.1 C.2
4. 从梯形上底的一点到下底引一条垂线，这点和垂足之间的线段叫梯形的(    )。 A.底 B.高 C.腰
5. 一个四边形只有一组对边平行，这个四边形是(    )。 A.平行四边形 B.梯形 C.长方形

### 四、画一画（共20分）
1. 画一个平行四边形，标出它的底和高。（8分）
2. 画一个梯形，标出它的上底、下底和高。（12分）

### 五、计算（共35分）
1. 平行四边形底8cm，高5cm，面积是多少？（10分）
2. 梯形上底6cm，下底10cm，高4cm，面积是多少？（10分）
3. 一块平行四边形菜地底15m，高8m，每平方米收菜3kg，共收菜多少kg？（15分）

## 参考答案
一、1.平行 2.梯形 3.无数 4.平行的两条边分别 5.垂线,高
二、1.× 2.× 3.√ 4.√ 5.√
三、1.B 2.A 3.B 4.B 5.B
四、（略）
五、1.40cm² 2.32cm² 3.360kg
