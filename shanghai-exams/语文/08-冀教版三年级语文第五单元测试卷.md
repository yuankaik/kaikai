## 第一部分 基础积累与运用（45分）

### 一、给红色的字选择正确的读音（3分）

1. 听到林则徐禁（jīn / **jìn**）烟的消息，人们禁（**jīn** / jìn）不住欢呼起来。
2. 爸爸又要出差（chā / **chāi** / chà）了，我难过得差（chā / chāi / **chà**）点哭起来。
3. 日本人那些罪恶（wù / **è**）的勾当，让我们中国人深恶（**wù** / è）痛绝。

### 五、选择恰当的词语填空（5分）

1. 做事要有（**耐心**），不能急躁。（耐心/忍耐）
2. （**因为**）今天下雨，（**所以**）运动会延期了。（虽然…但是…/因为…所以…）
3. 这次考试我没考好，但我（**并不**）灰心。

### 六、按要求改写句子（8分）

1. 这则寓言故事太有趣了！（改为反问句）
   **答案**: 这则寓言故事难道不是太有趣了吗？

2. 他的这种做法真是掩耳盗铃。（改为感叹句）
   **答案**: 他的这种做法真是掩耳盗铃啊！

## 第二部分 阅读理解（25分）

### 七、阅读《守株待兔》，回答问题

宋人有耕者。田中有株。兔走触株，折颈而死。因释其耒而守株，冀复得兔。兔不可复得，而身为宋国笑。

1. 解释下列加点字的意思（6分）
   - 兔**走**触株：**跑**
   - 因**释**其耒：**放下**
   - **冀**复得兔：**希望**

2. 翻译句子："兔不可复得，而身为宋国笑。"（4分）
   **答案**: 兔子再也没有得到，而他自己却被宋国人嘲笑。

3. 这个人为什么会被宋国人嘲笑？（5分）
   **答案**: 因为他把偶然发生的事情当成必然，妄想不劳而获，结果白白荒废了田地。

4. 这个故事告诉我们什么道理？（10分）
   **答案**: 不能把偶然的事情当成必然；不能死守狭隘经验，不知变通；幸福生活要靠劳动创造，不能指望不劳而获。
