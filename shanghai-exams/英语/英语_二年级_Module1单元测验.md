# 上海小学英语 - 二年级 Module 1 单元测验
## Oxford Shanghai Edition 2A Module 1

**时间：40分钟  满分：100分**

### Part 1 Listening（听力部分） 40分

**I. Listen and circle（听音圈出听到的单词）10分**
1. A. apple  B. banana  C. orange
2. A. cat  B. dog  C. bird
3. A. big  B. small  C. tall
4. A. red  B. blue  C. green
5. A. run  B. jump  C. walk

**II. Listen and number（听音排序）10分**
(   ) I can see a rabbit.
(   ) It is white.
(   ) It has long ears.
(   ) It can jump.
(   ) I like it.

**III. Listen and judge √/×（听音判断）10分**
1. The cat is big. (    )
2. The ball is red. (    )
3. I can swim. (    )
4. She is tall. (    )
5. He can run fast. (    )

**IV. Listen and choose（听问句选答句）10分**
1. A. I'm fine.  B. I'm five.  C. I'm a boy.
2. A. Yes, I can.  B. Yes, I am.  C. Yes, I do.
3. A. It's a dog.  B. They're dogs.  C. I like dogs.

### Part 2 Written（笔试部分） 60分

**I. Write the letters（写出字母的左右邻居）10分**
1. ___ B ___    2. ___ g ___    3. ___ M ___
4. ___ p ___    5. ___ T ___

**II. Choose（选择）20分**
1. — Can you swim? — Yes, I (    ).
   A. can  B. am  C. do
2. This (    ) a cat.
   A. is  B. are  C. am
3. — How old are you? — I'm (    ).
   A. fine  B. five  C. tall

**III. Read and match（连线）10分**
1. What's this?        A. I can jump.
2. Can you jump?       B. It's a rabbit.
3. How are you?        C. I'm fine.

**IV. Reading（阅读判断）20分**
I have a pet. It is a dog. It is brown. It has big eyes. It can run and jump. I like it very much.
1. I have a cat. (    )
2. The dog is brown. (    )
3. It has small eyes. (    )
4. It can run. (    )
5. I like my pet. (    )
