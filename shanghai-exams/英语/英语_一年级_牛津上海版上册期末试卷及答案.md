## Part 2 Reading and Writing（读写部分） 40%

### 一、Write the letters（写出下列字母的大小写） 10%

1. A → ______    6. G → ______
2. b → ______    7. h → ______
3. C → ______    8. I → ______
4. d → ______    9. j → ______
5. E → ______    10. f → ______

### 二、Look and match（看图连线，将图片与单词连接） 6%

1. 🍎  →  A. bag
2. 📕  →  B. apple
3. 🎒  →  C. cat
4. 🐱  →  D. book
5. ✏️  →  E. dog
6. 🐶  →  F. pencil

### 三、Choose the correct word（选择正确的单词填在横线上） 8%

1. I have a ______. (cat / cats)
2. This is my ______. (bag / bags)
3. I can see ______ birds. (two / a)
4. It's ______ apple. (a / an)
5. ______ is my mother. (She / He)
6. The flower is ______. (red / a red)
7. I ______ a girl. (am / is)
8. How many ______? (book / books)

### 四、Choose the best answer（选择最恰当的答案） 10%

(   ) 1. —Hello! I'm Tom. —______
A. Hello! I'm Alice.  B. Goodbye.  C. Thank you.

(   ) 2. —Give me a ruler, please. —______
A. Thank you.  B. Here you are.  C. Good morning.

(   ) 3. —What colour is it? —It's ______.
A. big  B. blue  C. a blue

(   ) 4. —Can you draw? —______
A. Yes, I am.  B. Yes, I can.  C. Yes, it is.

(   ) 5. —How many pencils? —______ pencils.
A. Five  B. A  C. One pencil

(   ) 6. I can see ______ elephant.
A. a  B. an  C. /

(   ) 7. This is ______ eye.
A. I  B. my  C. me

(   ) 8. Look ______ the bird.
A. at  B. on  C. in

(   ) 9. —Who is she? —______ is my mother.
A. He  B. She  C. It

(   ) 10. —Happy birthday! —______
A. Happy birthday!  B. Thank you.  C. Goodbye.

### 五、Read and draw (😊 or ☹)（读句子，判断正误，正确的画😊，错误的画☹） 6%
