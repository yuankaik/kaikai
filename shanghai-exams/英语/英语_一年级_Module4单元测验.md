# 上海小学英语 - 一年级 Module 4 单元测验
## Oxford Shanghai Edition 1A Module 4 The natural world

**时间：35分钟  满分：100分**

### Part 1 Listening 50分

**I. Listen and circle（听音圈出）10分**
1. A. bird  B. bee  C. butterfly
2. A. sun  B. moon  C. star
3. A. flower  B. tree  C. grass
4. A. big  B. small  C. beautiful
5. A. red  B. yellow  C. blue

**II. Listen and tick √/×（听音判断）10分**
1. The bird is red. (    )
2. I can see a butterfly. (    )
3. The flower is yellow. (    )
4. It's a sunny day. (    )
5. I like the tree. (    )

**III. Listen and number（听音排序）10分**
(   ) It's green.
(   ) It's tall.
(   ) Look at the tree.
(   ) I like it.
(   ) The tree is beautiful.

**IV. Listen and colour（听音涂色）10分**
1. colour the bird (    )
2. colour the flower (    )
3. colour the sun (    )

### Part 2 Written 50分

**I. Match（连线）12分**
1. bird      A. 花
2. flower    B. 鸟
3. tree      C. 太阳
4. sun       D. 树

**II. Read and choose（选择）18分**
1. — What can you see? — I can see (    ).
   A. a bird  B. bird  C. birds
2. The sun is (    ).
   A. red  B. yellow  C. blue
3. — What colour is the tree? — It's (    ).
   A. green  B. red  C. white

**III. Read and draw 😊/😢（阅读画表情）20分**
I can see a bird. It is blue. It is small. It can fly. It is beautiful. I like it.
1. The bird is blue. (    )
2. The bird is big. (    )
3. The bird can fly. (    )
4. I don't like it. (    )
