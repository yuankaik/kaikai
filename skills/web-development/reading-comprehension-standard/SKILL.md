# 阅读理解练习题质量标准

适用：上海小学四年级 | 用于审核网络爬取或自生成的阅读练习

## 六维标准

### 1. 短文标准
- **字数**：350—550字（中文汉字计数，不含标点）
- **原创性**：非教材课文，自主创作或改编
- **体裁**：记叙文/写人/写景/叙事/散文/书信优先；寓言、科普酌情

### 2. 主题要求
必须贴近以下四类之一：
- 校园生活
- 家庭亲情
- 日常生活
- 成长感悟

**排除**：纯寓言、纯科普说明文、脱离儿童生活的话题

### 3. 难度定位
中等偏基础——80%的四年级学生应能独立完成。用词不超纲，句式不复杂。

### 4. 题型结构（5题，每道题必须属于不同类型）

| 题号 | 题型 | 说明 | 示例问法 |
|------|------|------|---------|
| Q1 | 信息提取题 | 从文中直接找到答案 | "根据文章内容填空""文中写了哪些……" |
| Q2 | 词语理解题 | 理解词语在文中的意思 | "……是什么意思？""文中指什么？" |
| Q3 | 原因题 | 分析因果关系 | "为什么……""……的原因是什么？" |
| Q4 | 句子理解题 | 理解含义深刻的句子 | "这句话是什么意思？""文中画线句子……" |
| Q5 | 人物品质或中心题 | 概括人物品质或文章中心 | "……是一个怎样的人？""文章告诉我们什么道理？" |

### 5. 答案格式（每题必须包含4要素）

```
Q1. [题型：信息提取题]
标准答案：（完整答案文本）
段落依据：第X自然段
评分点：（每个得分点及分值）
常见错因：[标签1] [标签2]
```

**常见错因标签库**：
- 审题不清 / 信息找错段落 / 回答不完整
- 望文生义 / 脱离上下文 / 解释太笼统
- 因果倒置 / 只写结果不写原因 / 遗漏关键信息
- 理解停留在字面 / 没联系上下文 / 表达不清
- 概括太宽泛 / 找不到中心句 / 细节代替品质

### 6. 配套输出

**家长批改提示**：
```
📝 家长批改要点：
1. 重点检查Q___（指出最易错的题）
2. 如果孩子做错Q___，建议引导方法：___
3. 本篇核心能力：___（如"提取关键信息""理解人物情感"等）
```

**孩子重写任务**：
```
🔄 孩子重写任务：
错Q___→重新做这道题，用"答案模板"写
错Q___→回原文第___段再读一遍，用笔画出关键句后再答
错3题以上→全文重读+全卷重做
```

## 合规率计算

| 维度 | 权重 | 满分条件 |
|------|------|---------|
| 字数350-550 | 25% | 中文汉字在范围内 |
| 主题四选一 | 20% | 校园/家庭/生活/成长 |
| 难度适中 | 10% | 中等偏基础 |
| 5类题型 | 25% | 信息提取+词语+原因+句子+品质中心 |
| 答案格式 | 20% | 含段落定位+评分点+错因标签 |

≥80% = 合规，可直接使用。60-79% = 需修改。＜60% = 需重写。
