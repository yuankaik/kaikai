# 真题试卷搜索归档

## 工作流程

1. 创建归档目录：`~/exams/{语文,数学,英语}/`
2. delegate_task 并行三科搜索
3. 每科至少 3 个 subagent 并行
4. 提取内容 → MD 文件 + 答案
5. 持续搜索，不要轻易放弃

## 搜索源

| 网站 | 可用性 | 注意 |
|------|--------|------|
| 道客巴巴 doc88.com | 可搜预览 | 全文需付费 |
| 原创力 book118.com | 可搜预览 | 需登录 |
| 21 世纪教育网 | 资源多 | 需付费下载 |
| 百度 | 可用 | 有验证码 |
| Bing 国际版 | 不稳定 | 英文结果多 |

## 试卷类型

- 期末试卷 / 期中试卷 / 月考试卷
- 单元测验 / 阶段测验
- 名校真题（世外、上外附小、明珠小学等）
- 专项训练（拼音/阅读/时态/奥数）
- 毕业升学模拟卷

## MD 格式

```markdown
# [地区]小学[科目] - [年级] [类型]
## [题目部分]
### 一、[题型]
## 参考答案
```

## 归档结构

```
exams/
├── 语文/  (年级_类型_描述.md)
├── 数学/
└── 英语/
```

## 经验

- delegate_task 并行比串行快 3 倍
- 付费墙用 "基于大纲编写" 绕过
- 每份必须含完整答案
- 搜索 2 小时起，不要提前放弃

## 验证步骤

完成后必须验证：
```bash
# 检查缺答案的文件（排除作文/日记/范文）
find exams/ -name "*.md" | while read f; do
  grep -q "答案\|Answer\|answer\|写作要点\|总结" "$f" || echo "缺: $f"
done
```

## 注意事项

- **作文/日记/书信/读后感类文件天然不需要"答案"**，验证时跳过
- doc88/book118 提取的试卷经常缺答案，需手动补充
- 所有试卷类文件必须有题有答案，缺一不可
- 3/4/5 年级优先级最高（用户明确要求）
