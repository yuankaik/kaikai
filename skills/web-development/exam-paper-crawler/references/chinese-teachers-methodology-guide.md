# 中国小学语文名师资料搜索与整理指南

## 适用场景

用户要求搜索特级教师/优秀教师/骨干教师/名师的教育方法、教学实操、提分技巧等非试卷类教育资料时触发。

## 搜索流程

### 第一步：确定教师名单

常见搜索目标：贾志敏、乔亚梦、武凤霞、于永正、窦桂梅、丁有宽、王立根、颜思笠、支玉恒、李镇西、于漪等。

### 第二步：多源并行搜索（delegate_task 3路并行）

**路线A：百度百科为主**
- 直接访问 baike.baidu.com 搜索教师姓名
- 优点：无反爬、信息权威、含教学特色详情
- 抓取：简介、教育理念、教学特色、荣誉称号

**路线B：当当网书籍目录**
- search.dangdang.com 搜索"教师名+作文/阅读"
- 优点：获取完整书籍目录、教学模块结构、ISBN
- 抓取：章节结构、教学模块名称、训练专题列表

**路线C：道客巴巴/原创力教学实录**
- doc88.com 或 book118.com 搜索"教师名+教学实录/课堂实录"
- 注意：需登录、有付费墙
- 如无法访问，用路线A+B成果 + 专业知识补充

### 第三步：整理成结构化MD

每位教师五部分结构：
1. 简介（出身、经历、荣誉）
2. 核心理念（教育思想）
3. 实操方法（具体教学步骤、训练技巧）
4. 代表作（书名、出版社、年份）
5. 适合年级

### 第四步：生成PDF（可选）

```bash
pip install --break-system-packages weasyprint
python3 .hermes/skills/web-development/exam-paper-crawler/scripts/md2pdf.py input.md output.pdf
```

## 常见反爬与对策

| 网站 | 障碍 | 对策 |
|------|------|------|
| 百度搜索 m.baidu.com | 验证码弹窗 | 改用百度百科 baike.baidu.com 直达 |
| 搜狗/360搜索 | 安全检测拦截 | 放弃，改用百度百科+当当 |
| 道客巴巴 doc88.com | 需登录看全文 | 从搜索结果摘要提取关键信息 |
| 原创力 book118.com | 404或需登录 | 备选，优先用百度百科 |
| 知乎 | 反爬严格 | 避免使用 |
