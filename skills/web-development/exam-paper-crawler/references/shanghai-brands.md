## 3/4/5 年级优先级（用户指定）

用户明确要求三四五年级资源优先级最高。所有搜索和生成优先覆盖这三个年级。

## 文件命名规范

```
[品牌]_[年级]_[类型]_[描述].md
例：学而思_三年级_大白本精选.md
例：新东方_五年级_小升初衔接.md
```

## 作文类特殊处理

作文/日记/书信/读后感类文件天然不需要"答案"字段。验证时不应标记为缺陷。
