# 上海小学试卷生成模板

## 三科标准格式

### 语文
```markdown
# [品牌]上海小学语文 - [年级] [类型]
## 一、看拼音写词语（X分）
## 二、选择/判断（X分）
## 三、阅读理解（X分）
## 四、作文（X分）
## 参考答案
```

### 数学
```markdown
# [品牌]上海小学数学 - [年级] [类型]
## 一、口算（X分）
## 二、填空（X分）
## 三、判断/选择（X分）
## 四、计算（X分）
## 五、应用题（X分）
## 参考答案
```

### 英语
```markdown
# [品牌]上海小学英语 - [年级] [类型]
## Part 1 Listening（X分）
## Part 2 Written（X分）
## 参考答案
```

## 年级优先级
| 优先级 | 年级 |
|--------|------|
| 🔴 最高 | 三、四、五年级 |
| 🟡 中等 | 一、二、六年级 |

## 品牌模板前缀
- 学而思：`学而思_`
- 新东方：`新东方_`
- 精锐教育：`精锐_`

## 命名规范
```
[品牌]_[年级]_[描述].md
例: 学而思_三年级_计算小超市.md
```
