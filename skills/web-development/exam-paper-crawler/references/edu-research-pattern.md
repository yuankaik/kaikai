# 教育资料研究搜索模式

不只是试卷，当 kai哥需要**教育理念、学习方法、名师经验**等咨询类内容时，用以下模式。

## 并行搜索策略

将研究主题拆成 2-3 个子任务，用 `delegate_task` 并行执行：

```
task 1: 具体方法/技巧搜索（搜 book118/doc88 上的教案/实录）
task 2: 相关书单/课标/政策搜索（搜分级阅读、课标要求）
task 3: 名师人物搜索（搜百度百科 + 名师教学理念）
```

## 可靠搜索源

| 源 | 用途 | 技巧 |
|---|------|------|
| **book118.com（原创力文档）** | 教案、课堂实录、教学研究 | 搜索摘要可见，全文需付费但不影响提取要点 |
| **doc88.com（道客巴巴）** | 教师名单、教学资料 | 同 book118 |
| **百度百科** | 名师传记、教学理念 | 直接用 browser 抓取词条，稳定无反爬 |
| 百度搜索 | 综合搜索 | 触发验证码时跳过，用上面三个替代 |
| Bing/搜狗/360 | 备用 | 大多触发反爬，不作为主力 |

## 反爬应对

- 百度/Bing/搜狗大概率触发验证码或 WAF → **不要死磕，直接用 book118/doc88/百度百科**
- book118/doc88 不需要登录，搜索页面的摘要已足够提取核心信息
- 搜不到完整内容时，结合专业知识补充完善，注明"基于搜索结果摘要 + 专业知识整合"

## 输出格式

教育咨询类内容直接输出结构化的 Markdown 文章：
- 标题用 #、##、### 层级
- 列表用 - 或数字
- 表格用于对比信息
- 引用用 > 标注重要观点
- 末尾注明搜索来源和时间

保存路径：`~/shanghai-exams/{科目}/` + 同步到桌面 `C:\Users\admin\Desktop\shanghai-exams\{科目}/`

## 示例

本次搜索"上海小学语文特级教师阅读学习方法"：
- task 1 → 搜 book118/doc88 获贾志敏教学实录、阅读策略
- task 2 → 搜分级阅读、课标要求（百度反爬严重，部分失败）
- task 3 → 百度百科抓取贾志敏/于漪/窦桂梅/支玉恒/李镇西五位名师详情
- 输出两份 MD：综合资料汇编 + 名师详解
