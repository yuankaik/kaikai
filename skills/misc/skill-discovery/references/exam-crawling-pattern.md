# 教育内容爬取 — 上海小学真题案例

## 场景
用户要求搜索上海小学语数外真题试卷及答案，生成 MD 文件归档。

## 执行模式

1. **创建文件夹**：`~/shanghai-exams/{语文,数学,英语}/`
2. **并行搜索**：用 `delegate_task` 三科同时搜，每科至少 3 轮
3. **多轮补缺**：首轮覆盖主要年级后，第二轮补低年级和缺失学期
4. **持续搜索**：用户明确要求"不要放弃"，不断搜索直到覆盖全年级
5. **批量生成**：网络爬取失败时，用 `execute_code` 基于大纲批量编写

## 搜索来源
- 百度文库 (wenku.baidu.com) — 需登录/付费
- 道客巴巴 (doc88.com) — 预览片段可用
- 21世纪教育网 (21cnjy.com) — 资源多但需付费
- 原创力文档 (max.book118.com) — 需登录
- 菁优网 (jyeoo.com) — 初中以上为主

## 付费墙应对

国内教育网站几乎全需付费。应对策略：
1. 搜索预览 → 提取可见题目
2. 基于大纲 + 已知题型 → 编写完整试卷
3. 标注来源（如 `_book118` `_doc88` 后缀）

## 资料类型

| 类型 | 是否需要答案 | 说明 |
|------|-------------|------|
| 期末/期中/月考试卷 | ✅ 必须 | 每题配答案 |
| 名校真题 | ✅ 必须 | 每题配答案 |
| 口算心算 | ✅ 必须 | 末尾附答案 |
| 辅导资料（大白本等） | ✅ 必须 | 每题详解 |
| 优秀作文 | ❌ 不需要 | 范文参考，天然无答案 |
| 日记/书信/读后感 | ❌ 不需要 | 范文参考 |

## 验证命令

```bash
# 检查试卷类缺答案（排除作文/日记）
find exams/ -name "*.md" ! -path "*优秀作文*" | while read f; do
  grep -q "答案\|Answer\|answer" "$f" || echo "缺答案: $f"
done

# 检查几乎空文件
find exams/ -name "*.md" | while read f; do
  [ $(wc -l < "$f") -lt 10 ] && echo "几乎空: $f ($(wc -l < $f)行)"
done
```

## MD 格式模板

### 试卷类
```
# [品牌/地区] - [年级] [类型]
## 一、[题型]（分值）
## 二、[题型]
...
## 参考答案
```

### 口算类
```
# 小学[X]年级口算100题
## 计时：___分___秒  得分：___
1. XX + XX =    2. ...
## 参考答案
```

### 作文类
```
# 作文题目：[标题]
## 范文
[正文]
## 点评
优点/不足
```

## 目录结构
```
exams/
├── 语文/        (真题试卷)
├── 数学/        (真题 + 名校 + 奥数)
├── 英语/        (真题 + 专项)
├── 学而思/      (大白本/计算超市/阅读/作文/英语)
├── 新东方/      (数学/语文/英语)
├── 精锐教育/    (数学/英语)
├── 其他教育品牌/ (昂立/四季/乐课力/绿光)
├── 口算心算/    (100题/份, 1-6年级)
└── 优秀作文/    (范文, 无答案)
```
