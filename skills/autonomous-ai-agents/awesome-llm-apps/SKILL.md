# Awesome LLM Apps

100+ 可直接运行的 AI Agent & RAG 应用模板，Apache 2.0 开源。

## 快速上手

```bash
git clone https://github.com/Shubhamsaboo/awesome-llm-apps.git
cd awesome-llm-apps/<分类>/<应用名>
pip install -r requirements.txt
streamlit run <app>.py
```
