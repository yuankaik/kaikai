# 文玩仙图/滤镜识别

## 常见造假手法

### 灯光造假
- 黄色暖光 → 让普通核桃显"红"
- 侧光 → 突出纹理，掩盖瑕疵
- 多灯位 → 消除阴影，让平面显立体

### 滤镜造假
- 饱和度拉高 → 颜色鲜艳不自然
- 对比度拉高 → 纹理过度锐化
- 色温偏暖 → "老物件"感

### 后期处理
- 背景虚化 → 掩盖边缘瑕疵
- 局部提亮 → 突出"金星""包浆"
- 视频抽帧 → 去掉展示瑕疵的画面

## AI 检测方法

### 色彩分析
```python
# 检测过度饱和
from PIL import Image
import numpy as np

img = Image.open("photo.jpg")
arr = np.array(img)

# 红色通道异常高
r_mean = arr[:,:,0].mean()
if r_mean > 180:  # 红通道过曝
    print("⚠️ 疑似暖光/滤镜：红色通道均值%{r_mean}")
```

### 高光检测
- 多处不自然高光 → 多灯位打光
- 高光面积占比 > 15% → 过度曝光

### 视频分析
- 帧间颜色突变 → 后期调色
- 物体边缘抖动 → 后期处理痕迹

## 对策
1. 要求自然光下拍摄
2. 要求放参照物（硬币/手掌）
3. 要求拍视频而非照片
4. 多角度对比

## 红黄绿信号
- 🟢 色彩直方图自然分布，高光合理
- 🟡 暖色调偏重但不过度，建议自然光复拍
- 🔴 饱和度异常、多处高光、帧间颜色突变
