# 手串鉴定指南

## 材质分类
| 材质 | 特征 | 造假方式 |
|------|------|----------|
| 小叶紫檀 | 牛毛纹、金星、沉水 | 红酸枝冒充、金粉造假 |
| 黄花梨 | 鬼脸纹、降香味 | 缅甸花梨冒充 |
| 沉香 | 油线、密度低、香味 | 泡油、压缩 |
| 金刚菩提 | 瓣数、肉度、桩型 | 落地红冒充、做旧 |

## 快速辨别

### 小叶紫檀
- 白纸划痕：真→橘红色，假→无/其他色
- 沉水测试：真→秒沉
- 酒精测试：真→渗出红色
- 金星：真金星在牛毛纹里断续分布，假金星连续均匀

### 黄花梨
- 鬼脸纹自然不规则
- 降香味独特，其他花梨无此味
- 荧光反应：浸泡后水面有蓝色荧光

### 沉香
- 真沉香轻，高仿沉香重（压缩过）
- 真沉香香味淡雅持久，假的一阵就没了
- 油线自然晕开，假的边界生硬

### 金刚菩提
- 真金刚有天然突起"奶嘴"
- 落地红瓣数少、纹路单调
- 做旧金刚颜色暗沉不自然

## 图片鉴定要点
1. 微距拍材质纹理
2. 自然光拍颜色
3. 拍孔道（工艺痕迹）
4. 对比参照物（看大小比例）

## 红黄绿信号
- 🟢 纹理自然、有特征纹路、重量正常
- 🟡 颜色统一得好得不自然，需实物确认
- 🔴 纹理机械感、重量异常、化学气味
