# 文玩鉴定助手

支持：文玩核桃、手串珠串、玉石翡翠。分析照片/视频，输出风险等级。

## 使用方式

用户微信发图/视频 → 自动加载本技能 → 返回鉴定报告。

## 图片分析方案

### 优先方案：Ollama 本地视觉模型（免费）

已安装 `minicpm-v:8b`，Windows 后台运行中。
```bash
# 本地 AI 视觉分析（免费、离线）
python3 scripts/ollama_vision.py /path/to/image.jpg
```

### 辅助方案：PIL 色彩检测

```bash
# 色彩/滤镜/高光分析
python3 scripts/image_check.py /path/to/image.jpg
```

### 最佳流程
```
1. image_check.py → 检测仙图/滤镜（色彩分析）
2. ollama_vision.py → AI 视觉分析（品类识别、特征判断）
3. 对照 knowledge/ 知识库
4. 输出红黄绿风险报告
```

## 鉴定流程

1. **仙图检测** → 先判断照片是否被灯光/滤镜处理过
2. **品类识别** → 核桃/手串/玉石
3. **特征分析** → 对照知识库检查关键特征
4. **输出报告** → 红黄绿风险等级 + 详细说明

## 输出格式

```
🔍 文玩鉴定报告

📸 图像质量：🟡 暖光偏重，建议自然光重拍
🪨 初步判断：核桃 — 疑似四座楼
🔴 风险点：
  - 纹理有机械重复感
  - 底部无天然毛孔
🟢 可信特征：
  - 桩型符合四座楼特征

⚠️ 综合风险：🔴 高风险，建议实物上手
```

## 知识库

- `knowledge/核桃鉴定.md` — 品种、真假、做旧
- `knowledge/手串鉴定.md` — 材质、工艺、冒充
- `knowledge/玉石鉴定.md` — 翡翠ABC、和田玉
- `knowledge/仙图识别.md` — 灯光/滤镜/视频分析

## 局限性

- 无法替代实物鉴定
- 强光/滤镜下准确度下降
- 高端仿品需专业仪器
- 以辅助参考为目的，不做最终判断
