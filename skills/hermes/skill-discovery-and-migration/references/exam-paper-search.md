# 教育真题搜索工作流

## 场景

搜索上海小学生语数外真题试卷（月考、期末、单元测验、模拟考、名校真题等），生成 MD 文件归档。

## 搜索源

| 平台 | 可用性 | 备注 |
|------|--------|------|
| 道客巴巴 doc88.com | 部分可用 | 搜索结果预览可看到部分题目，完整下载需付费 |
| 原创力文档 book118.com | 有限 | 需登录，有反爬 |
| 21世纪教育网 21cnjy.com | 有限 | 大量资源但需付费下载 |
| 百度文库 wenku.baidu.com | 有限 | VIP文档占多数 |
| 菁优网 jyeoo.com | 有限 | 主要覆盖初中以上 |

## 策略

1. **先用浏览器搜**：直接在 doc88/book118 搜索关键词
2. **提取预览**：用 browser_console 提取搜索结果的预览文本
3. **基于大纲补全**：即使无法下载完整试卷，也可以基于沪教版大纲编写标准试卷
4. **并行搜索**：三科用 delegate_task 同时搜

## 试卷格式

每份 MD 文件包含：
- 试卷标题（科目 + 年级 + 类型）
- 考试时间和分值
- 各题型（选择/填空/计算/阅读/写作）
- 完整参考答案

## 关键词

- 通用：`上海小学[科目]期末试卷 真题`
- 月考：`上海小学[科目]月考真题`
- 单元：`上海小学[科目]第X单元测试`
- 名校：`[学校名]小学[科目]试卷`
- 专项：`上海小学[科目][专项]训练卷`
